Términos de uso

Este proyecto, PortfolioLab (portfoliolab.ddnsfree.com), es un desarrollo de código abierto. El objetivo es que cualquier persona pueda aprovechar las ideas, el código y la estructura de esta web para sus propios proyectos.

Condiciones

  • Puedes utilizar, modificar y distribuir el código sin coste alguno.
  • Puedes emplearlo tanto para proyectos personales como comerciales.
  • La única condición es que, en algún lugar visible de tu proyecto (código fuente, documentación o interfaz), incluyas una mención de la autoría original.

Licencia recomendada

El código se publica bajo una licencia de estilo MIT, lo que significa que se otorga libertad total de uso, siempre que se mantenga el aviso de copyright y se reconozca la autoría original.

“El software se proporciona ‘tal cual’, sin garantía de ningún tipo, expresa o implícita, incluyendo, pero no limitado a, garantías de comerciabilidad o adecuación para un propósito particular.”

Si reutilizas este código, por favor incluye un enlace a este sitio web o una referencia a PortfolioLab.
